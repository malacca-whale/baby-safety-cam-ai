# Baby monitor
이 프로그램은 아기 침대 근처에 설치된 카메라를 통해 아기를 모니터링하고, 아기가 울거나 뒤집거나 하는 등의 행동을 감지하면 사용자에게 알림을 보내는 프로그램입니다.
ollama vision 모델을 사용하여 아기의 상태를 감지합니다.
ollama 에 요청을하고 받을땐 structured model output 을 사용하여, 검사하고자 하는 항목들에 대해 검사하여 json 형태로 반환받습니다.
Discord 에 연결하여, 두개의 채널에 알림을 전송합니다.

1번째 채널: 아기 경고사항 발생시 알림
* 아기의 얼굴에 천이 덮여서 질식상황이면 경고
* 뒤집어져서 질식 위험이 있으면 경고
* 아기가 침대 밖으로 나가는 위험이 있으면 경고

2번째 채널: 5분에 한번씩 카메라를 통하여 파악된 아기의 상태변화를 요약하여 전송한다.
* 아기는 어떤 상황인지.
* 아기에게 움직임이 있었는지 (optical flow 활용)
* 아기의 자세가 어떻게 변했는지
* 아기의 숨소리가 들렸는지
* 카메라를 통하여 캡쳐를 하는것은 더 많이 진행하되, 요약하여 전송을 5분에 한번씩 하도록 한다.

파이썬을 통하여 개발하고, uv 를 사용하도록 하라.
웹페이지를 만들어서 카메라 영상과 음성을 실시간으로 보고 들을 수 있게 하라.
복수의 카메라와 마이크 중 사용할 카메라와 마이크를 선택할 수 있게 하라.

AI 관련
vision model 은 qwen3-vl 을 사용한다. https://ollama.com/library/qwen3-vl
LLM 또한 vision model 은 qwen3-vl 를 사용하도록 한다.
아기 음성을 감지하는 모델은 허깅페이스에서 적절한 모델을 다운받아 적용하도록 한다.
