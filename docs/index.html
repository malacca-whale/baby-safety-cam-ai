주요 기능

스마트한 아기 모니터링

최신 AI 기술을 활용하여 아기의 안전을 종합적으로 모니터링합니다

👁

비전 AI 분석

Ollama Qwen3-VL 모델이 아기의 자세, 얼굴 가림, 침대 이탈을 실시간으로 분석하고 위험도를 평가합니다.

🏃

움직임 감지

Lucas-Kanade Optical Flow 알고리즘으로 아기의 움직임을 15FPS로 추적하고 패턴을 분석합니다.

🎤

오디오 분석

Librosa를 활용한 스펙트럼 분석으로 아기 울음과 호흡 패턴을 실시간으로 감지합니다.

💬

Discord 알림

위험 상황 발생 시 즉시 경고 알림과 캡처 사진을 Discord로 전송합니다. 5분마다 상태 보고서도 제공됩니다.

⚡

실시간 스트리밍

MJPEG 비디오와 WebSocket 오디오 스트리밍으로 어디서든 실시간으로 아기를 확인할 수 있습니다.

🗃

데이터 기록

SQLite 데이터베이스에 모든 분석 결과와 이벤트가 저장되어 히스토리를 확인할 수 있습니다.

작동 방식

간단한 4단계 설정

몇 분 만에 AI 아기 모니터링 시스템을 시작할 수 있습니다

1

프로젝트 설치

GitHub에서 프로젝트를 클론하고 uv로 의존성을 설치합니다. Python 3.11+와 Ollama가 필요합니다.

2

AI 모델 다운로드

Ollama를 통해 Qwen3-VL 비전 모델을 다운로드합니다. 한 줄 명령어로 완료됩니다.

3

Discord 웹훅 설정

Discord 서버에 웹훅을 만들고 .env 파일에 URL을 입력합니다. 경고용과 상태 보고용 두 개의 채널을 사용합니다.

4

모니터링 시작

서버를 실행하고 웹 브라우저에서 대시보드에 접속합니다. 이제 AI가 아기를 24시간 모니터링합니다.

스크린샷

직관적인 인터페이스

깔끔하고 사용하기 쉬운 웹 대시보드를 제공합니다

웹 대시보드

실시간 비디오, 상태 카드, 히스토리를 한눈에 확인
